# ShiftPally

Paladin buff manager for WoW TBC Classic Anniversary. A replacement for PallyPower.

## Why not PallyPower?

- Simpler, more intuitive interface
- Easier to see what's assigned at a glance
- Less mental overhead for buff refresh
- Adds quality-of-life features

## Features

- **One-click buff button** — queues all missing blessings, aura, and Righteous Fury. Just keep clicking.
- **Three views** — Class (set blessings per class), Individual (per player), and Overrides (see exceptions at a glance)
- **Role-aware quick setup** — "Salv Defaults" and "Class Defaults" buttons auto-assign the whole group based on tank/healer/DPS roles in one click
- **Pet blessing assignments** — optionally show hunter/warlock pets for individual buff assignments
- **PallyPower compatible** — works seamlessly in groups with PallyPower users, no migration needed

## Usage

Type `/sp` to toggle the bar. Click **Edit** to open the assignment panel. Click **Buff** to cast the next missing buff.

## Install

Copy the `ShiftPally` folder into your `Interface/AddOns/` directory.
